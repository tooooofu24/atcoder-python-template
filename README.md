## Python で Atcoder やるとき用

Python で Atcoder 出るときの雛形

## ファイル説明

| ファイル名 | 説明                           |
| ---------- | ------------------------------ |
| answer.py  | 回答になるソースコード         |
| input.txt  | 標準入力ファイル               |
| run.sh     | 回答を実行するシェルスクリプト |
| memo.md    | メモ                           |

## 使い方

回答の実行
入力を`input.txt`に書き、`answer.py`にコードを記述して以下を実行。

```bash

$ ./run.sh

```
